# 💰 Finance Dashboard UI (Angular 19)
- Bootstrap (UI)
- ApexCharts (Data Visualization)
- RxJS (State Management)

---

## 📂 Project Structure

src/app/
├── models/
├── services/
├── components/
│ ├── dashboard/
│ ├── transactions/
│ ├── insights/
│ ├── header/
│
├── app.component.ts
├── app.routes.ts

---

## ⚙️ Installation & Setup

```bash
npm install
npm install bootstrap apexcharts ng-apexcharts
ng serve
```

Run at: http://localhost:4200
